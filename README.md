# 🧑‍💻 My Portfolio (Next.js + Tailwind CSS)

Đây là dự án portfolio cá nhân được viết lại từ HTML tĩnh, sử dụng Next.js và Tailwind CSS để tối ưu hơn về performance và dễ mở rộng về sau.
